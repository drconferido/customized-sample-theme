# Customized Sample Theme

A modular, OOP-based WordPress theme built from scratch.  
This project demonstrates custom post types, template hierarchy, and responsive design for practice and learning.

## Installation
1. Download or clone this repository into your `wp-content/themes/` directory.
2. Activate the theme from the WordPress admin dashboard.

## Features
- Modular structure with Object-Oriented PHP
- Custom Post Type: Urban Legends
- Responsive design
- Basic template hierarchy
- Editable styles and layout

## Usage
Modify `style.css` and `functions.php` to experiment with WordPress theme development.  
Extend classes and modules to add new functionality.
